# Vision

LifeSession is an open-source operational terminal for maintainers and small teams.

Modern teams do not only need task lists. They need a way to understand what
happened, why decisions were made, which events mattered, and what should be
reviewed next. LifeSession packages that work into sessions.

## Core Idea

A session is a bounded unit of operational attention:

- a mission to pursue;
- events that arrive during the mission;
- decisions made by humans or agents;
- state changes that can be reviewed later;
- a close record that summarizes the outcome.

## Non-Goals

- LifeSession is not a private operations dump.
- LifeSession is not live trading software.
- LifeSession is not a replacement for human review.
- LifeSession is not tied to a specific messaging, database, or cloud provider.
