# Roadmap

## Phase 0 — Public Skeleton

- Project docs and public/private boundary.
- JSON schemas.
- Fake examples.
- Demo Python package.
- CI and contribution stubs.

## Phase 1 — Local Session Runtime

- Validate example sessions against schemas.
- Add local session replay.
- Add text dashboard rendering.

## Phase 2 — Maintainer Workflows

- Issue triage session templates.
- Decision review helpers.
- Session close summaries.

## Phase 3 — Integrations

- Optional adapters for public, user-configured tools.
- Strict boundary checks before any adapter ships.
